# AI-Powered Medical Assistant

An AI-powered healthcare assistant built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

## Features
- AI-powered medical query assistance
- Secure user authentication
- Medical history management
- Responsive UI
- MongoDB database integration
- REST API architecture

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication
